### About the project
Space Explorer is a browser-based 3D space game built with HTML, CSS, and JavaScript using A-Frame. The main menu lets you set a seed for deterministic planet layouts, then you fly through space to discover worlds, land, and collect coins. Coins persist per planet via cookies, points are tracked across the session, and a 5-minute timer ends the run on a game over screen where you can reset and return to the menu.
